A Chrome extension to help users find the best NGO or volunteer opportunity based on their interests.

## Problem
There are many NGOs and it's hard to know which one to choose or how to identify the right fit.

## Features
- Scour different websites to find opportunities based on user input
- Chatbot interface or strong filtering to narrow down options
- Later: allow organizations to submit their own opportunities
- Pop-up volunteer section — one-day events with no weekly commitment

## Ranking Analysis (Auto-rank)

Feasibility is moderate — scraping multiple NGO websites reliably is technically challenging and prone to breaking, and a Chrome extension format adds constraints around data fetching and storage. Time to build is high given the web scraping complexity, chatbot/filtering logic, and the multi-source data aggregation required. Originality is low since platforms like VolunteerMatch, Idealist, and All for Good already solve this problem, making the extension more of a UI layer than a novel solution.
