# DoorDash Dietary Filter Browser Extension

A browser extension for DoorDash that filters menu items based on dietary restrictions and preferences.

## Status

No longer actively pursuing this project. It was a good learning experience — got it working somewhat before stepping away.

## Problems Encountered

- **Rapid UI changes:** DoorDash updates its interface so frequently that keeping up with the changes is a serious maintenance challenge.
- **Unparsable HTML:** The page markup is essentially gibberish, making it very difficult to extract menu data in a reliable way.
- **Personal use case:** Even if both issues above were resolved, the motivation to maintain it is low — not a frequent DoorDash user personally.

## Reflection

Got it partially working before dropping the project, which was a satisfying outcome in itself. Not something being actively pursued anymore, so it has been left as-is for now.
