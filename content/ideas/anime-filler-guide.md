# Crunchyroll Filler Episode Marker — Chrome Extension

A free Chrome extension that scrapes data from Anime Filler List and displays filler episode information directly on Crunchyroll — no more flipping back and forth between tabs.

## Overview

Browse Crunchyroll and instantly see which episodes are fillers, right where you need them. Each episode is marked with a distinct icon so you can decide what to skip without ever leaving the platform. Works with all anime available on Crunchyroll.

## Features

- **Inline filler markers** — episodes are flagged directly on Crunchyroll with a ⭐ icon at a glance
- **Filler type breakdown** — fillers are split into three distinct categories, each with its own shape/icon:
  - **Filler** — standard filler episodes
  - **Mixed Canon** — episodes that blend canon and filler content
  - **Fan-Favorite** — beloved filler arcs (e.g. the G-8 arc in One Piece)
- **Up-to-date data** — AnimeFillerList data is updated to include the latest anime
- **Extension badge** — the Chrome extension badge displays the episode count at a glance
