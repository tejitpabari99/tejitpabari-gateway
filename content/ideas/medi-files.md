# Personal Medical File Organizer

A personal medical file organizer to store, organize, and access medical records, prescriptions, and health history in one place.

---

## Status

This project is complete!

> ⚠️ **Note:** A test user is currently set up, which means any file uploaded can be seen by anyone. Be mindful of what you upload in this environment.

---

## Features

- **Upload medical documents** — Add your medical records, prescriptions, and other health-related files.
- **View uploaded documents** — Browse and access all previously uploaded files in one place.
- **Search through medical data** — Query your medical data and receive both the relevant information and a direct link to the corresponding document.
